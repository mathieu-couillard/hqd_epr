** Note on Maintenance
This repository is a static archive of scripts developed during my doctoral research at OIST. It is provided "as-is" for academic review.

* Overview
=hqd_epr= provides a full pipeline for experimental quantum information physics, from hardware-level data collection to sophisticated physical modeling of spin ensembles.

* Repository Structure
- =acquisition/= :: Scripts for automated data collection.
  - Interfacing with VNAs for frequency sweeps.
  - Automated Magnetic field (B-field) scanning and gradient control.
  - Raw data logging to CSV format.
- =analyze_data/= :: Core analysis logic.
  - =fit_2d_spectrum.py= : Fits 2D data using Dissipative Coupled Mode Theory.
  - =resonator_characterization.py= : Extracts $Q$ factors and cavity parameters.
  - =utils.py= : Physical utility functions.

* Dependencies
- =python= (>= 3.9)
- =numpy=, =scipy=, =pandas=, =matplotlib= :: The scientific computing stack.
- =pyvisa= :: For instrument control (GPIB/Ethernet) in the =acquisition= scripts.
